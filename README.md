# Open Slidysim

[Slidysim](https://slidysim.github.io/) leaderboard page

Welcome to **Open Slidysim**, a community-driven sliding puzzle leaderboard. After the original Slidysim leaderboard shut down, we created this space to keep the fun going and make sure everyone has a place to track their achievements.
